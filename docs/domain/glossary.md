# Glossário (Domínio Brikick)

- **Catalog Item**: entidade do catálogo (Part/Set/Minifig/Book/Gear/etc.) com ID canónico.
- **Color**: cor (mapeamento/IDs) para itens que variam por cor.
- **Inventory Item**: item em posse de um seller (stock “real” do seller).
- **Listing**: anúncio/publicação (o que aparece no marketplace).
- **Order**: encomenda criada a partir do checkout.
- **Shipment**: envio associado a uma order (tracking, carrier, estado).
- **Dispute/Claim**: processo de reclamação (buyer/seller) com evidências e decisão.
- **RBAC**: permissões baseadas em roles (buyer/seller/admin/mod).
- **BOID**: BrickOwl internal ID (pode faltar; tratar como BL-only quando ausente).
